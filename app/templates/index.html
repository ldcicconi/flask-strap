Welcome to flask-strap!

flask-strap is a simple place to get started when developing any user-driven website with python.

flask-strap was not built by one individual. Rather, it is a compilation of various python, HTML5, and CSS3 libraries and code-snippets scattered across the web.

Here are some of the libraries and packages used:

  • Flask: a python microframework
  • Bootstrap: a responsive front-end framework
  • Flask-SQLAlchemy: an easy way to manage relational databases
  • Flask-WTForms: a simple solution to managing web-forms
  • Flask-Login: an intuitive way to log users in and out
  • Flask-Bcrypt: a way to hash passwords for safe storage

Here are sources from which code was generously provided:

  • "The Flask Mega-Tutorial" by Miguel Grinberg

    • Much of flask-strap is based on the Mega-Tutorial, including views, models, and templates. Great place to learn Flask.

  • "Login & Signup forms in panel" by calvinko

    • A great login/signup form

  • "Lightweight Python Apps with Flask, Bootstrap, and Heroku" by Ryan Shea

    • A simple yet effective tutorial on using Bootstrap, git, Heroku, and Flask
